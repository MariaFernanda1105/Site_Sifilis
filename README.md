# 📌 Site Sífilis

Projeto de site informativo sobre a Sífilis, desenvolvido para apoiar iniciativas do Governo do Tocantins.
A plataforma tem como objetivo conscientizar a população por meio de informações claras e acessíveis sobre sintomas, formas de transmissão, prevenção e tratamento, além de incentivar a busca por atendimento médico adequado.

# 🎯 Objetivo

Promover conscientização sobre a doença
Facilitar o acesso à informação confiável
Incentivar a procura por serviços de saúde.

# 🧪 Qualidade e Testes (QA)

Durante o desenvolvimento, foram aplicadas práticas de Controle de Qualidade, como:
